# DevSecOps Lab

My structured 24-week self-study journey into DevSecOps — built from the ground up.

## Structure

Each week has its own folder containing scripts, configs, and notes.

| Week | Topic | Status |
|------|-------|--------|

- Land a DevSecOps role
